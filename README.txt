***************
 IPDA Web Site
***************

This is the software configuration for the website of the International
Planetary Data Alliance (IPDA).  The IPDA website is normally available at
http://planetarydata.org/.

This package contains a Python_ based buildout that uses the Plone_ content
management system to run the IPDA website.

Installation
============

To set up this software, see the INSTALL.txt file.
